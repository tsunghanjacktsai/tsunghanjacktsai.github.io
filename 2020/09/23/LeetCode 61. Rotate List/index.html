LeetCode 61. Rotate List

Problem

Source link: Rotate List

Given a linked list, rotate the list to the right by k places, where k is non-negative.

Example 1:

Input: 1->2->3->4->5->NULL, k = 2
Output: 4->5->1->2->3->NULL
Explanation:
rotate 1 steps to the right: 5->1->2->3->4->NULL
rotate 2 steps to the right: 4->5->1->2->3->NULL

Example 2:

Input: 0->1->2->NULL, k = 4
Output: 2->0->1->NULL
Explanation:
rotate 1 steps to the right: 2->0->1->NULL
rotate 2 steps to the right: 1->2->0->NULL
rotate 3 steps to the right: 0->1->2->NULL
rotate 4 steps to the right: 2->0->1->NULL

Design

By observing examples above, we can find that the head node will always be replaced by the last one. Hence, we can utilize the feature of circular linked lists to solve this problem, which enables the last node connect with the head. The algorithm process is described below:

  1. We need to find out where the last node is and connect it with the head to form a circular list.
  2. Start construct the result list by marking its ending position out first.
  3. We then can get the head of the result list, which is lastNodeOfResultList.next.
  4. Obtain the result list by pointing its last node to null.

Implementation

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
  public ListNode rotateRight(ListNode head, int k) {
    if (k <= 0 || head == null || head.next == null) {
      return head;
    }

    ListNode lastNode = head;
    int len = 1;
    while (lastNode.next != null) {
      lastNode = lastNode.next;
      len++;
    }
    lastNode.next = head;

    k %= len;
    int skipLen = len - k;
    ListNode lastNodeOfResultList = head;
    for (int i = 0; i < skipLen - 1; i++) {
      lastNodeOfResultList = lastNodeOfResultList.next;
    }

    head = lastNodeOfRotatedList.next;
    lastNodeOfRotatedList.next = null;
    return head;
  }
}

Complexity

Time: $O(n)$

Space: $O(1)$
