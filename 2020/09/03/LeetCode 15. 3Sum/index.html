LeetCode 15. 3Sum

Problem

Source link: 3Sum

Given an array nums of n integers, are there elements a, b, c in nums such that a + b + c = 0? Find all unique triplets in the array which gives the sum of zero.

Notice that the solution set must not contain duplicate triplets.

Example 1:

Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]

Example 2:

Input: nums = []
Output: []

Example 3:

Input: nums = [0]
Output: []

Constraints:

  • 0 <= nums.length <= 3000
  • -105 <= nums[i] <= 105

Design

The problem requires us to find all combinations of three numbers that the sum of elements within each of it is equal to 0. This could be solved by applying two pointers pattern as below:

  1. Sort the array nums​ first for the binary search later.
  2. Traverse through the array and set up nums[i] as the first number of the result array.
  3. Initiate left and right pointers for the subsequent subarray, which the left pointer is at nums[i] + 1, and the right one is at nums.length - 1.
  4. Calculate the sum of nums[i], nums[left] and nums[right]. If the result is equal to 0, then return those three elements. Otherwise, move the left pointer forward or the right pointer backward while the sum is smaller or larger than 0.
  • Note: We need to make sure it does not recalculate the same value.

Implementation

class Solution {
  public List<List<Integer>> threeSum(int[] nums) {
    List<List<Integer>> res = new ArrayList<>();
    Arrays.sort(nums);
    for (int i = 0; i < nums.length - 2; i++) {
      if (i > 0 && nums[i] == nums[i - 1]) {
        continue;
      }
      int left = i + 1, right = nums.length - 1;
      while (left < right) {
        int sum = nums[i] + nums[left] + nums[right];
        if (sum == 0) {
          res.add(Arrays.asList(nums[i], nums[left], nums[right]));
          left++;
          right--;
          while (left < right && nums[left] == nums[left - 1]) {
            left++;
          }
          while (left < right && nums[right] == nums[right + 1]) {
            right--;
          }
        } else if (sum < 0) {
          left++;
        } else {
          right--;
        }
      }
    }
    return res;
  }
}

Complexity Analysis

Time: $O(n ^ 2)$

Space: $O(n)$
