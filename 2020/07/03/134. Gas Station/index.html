134. Gas Station

Source link: https://leetcode.com/problems/gas-station/

Design

The basic idea is that check if the car will run out of oil before reaching the next gas station, if so, recalculate the starting point.

  1. Travel through the cycle and get the gain by gas[i] - cost[i] and add it to the total gain.
  2. Rebase the starting point if the car will run out of oil before reaching the next station (cost > gain), else calculate the accumulation of gas inside the car.

Implementation

class Solution {
  public int canCompleteCircuit(int[] gas, int[] cost) {
    // // see if we can go through the whole cycle without running out of oil
    // int tank = 0;
    // for (int i = 0; i < gas.length; i++) {
    //   tank += gas[i] - cost[i];
    // }
    // if (tank < 0) {
    //   return -1;
    // }
    int start = 0;
    int accumulate = 0;
    int totalGain = 0;
    for (int i = 0; i < gas.length; i++) {
      int curGain = gas[i] - cost[i];
      totalGain += curGain; // calculate the total gain to see if we would run out of oil before finishing the cycle
      // if we will run out of oil before reaching the next station
      if (accumulate + curGain < 0) {
        start = i + 1; // rebase the starting point
        accumulate = 0;
      } else {
        accumulate += curGain;
      }
    }
    return totalGain >= 0 ? start : -1;
  }
}

Complexity Analysis

Time: $O(n)$

Space: $O(1)$
