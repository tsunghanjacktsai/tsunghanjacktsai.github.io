LeetCode 16. 3Sum Closest

Problem

Source link: 3Sum Closest

Given an array nums of n integers and an integer target, find three integers in nums such that the sum is closest to target. Return the sum of the three integers. You may assume that each input would have exactly one solution.

Example 1:

Input: nums = [-1,2,1,-4], target = 1
Output: 2
Explanation: The sum that is closest to the target is 2. (-1 + 2 + 1 = 2).

Constraints:

  • 3 <= nums.length <= 10^3
  • -10^3 <= nums[i] <= 10^3
  • -10^4 <= target <= 10^4

Design

The problem requires us to find the triplet that its sum of elements is the most closest to the given target. Same as solving 3Sum, we can apply the left & right pointers approach.

  1. Set up the first element of a triplet in each iteration.
  2. Try each sum of combinations with any other two elements in the array. Return the smallest difference against the target.

Implementation

class Solution &#123;
  public int threeSumClosest(int[] nums, int target) &#123;
    Arrays.sort(nums);
    int minDiff = Integer.MAX_VALUE;
    for (int i = 0; i < nums.length - 2; i++) &#123;
      int left = i + 1, right = nums.length - 1;
      while (left < right) &#123;
        int diff = target - nums[i] - nums[left] - nums[right];
        if (diff == 0) return target - diff;
        if (Math.abs(diff) < Math.abs(minDiff)) &#123;
          minDiff = diff;
        &#125;
        if (diff > 0) &#123;
          left++;
        &#125; else &#123;
          right--;
        &#125;
      &#125;
    &#125;
    return target - minDiff;
  &#125;
