LeetCode 19. Remove Nth Node From End of List

Problem

Source link: Remove Nth Node From End of List

Given a linked list, remove the n-th node from the end of list and return its head.

Example:

Given linked list: 1->2->3->4->5, and n = 2.

After removing the second node from the end, the linked list becomes 1->2->3->5.

Note:

Given n will always be valid.

Design

Two pass solution

This is an intuitive method, which the main process is described below:

  1. Get the length of the list first for the following calculation.
  2. Find the length - nth node and delete it from the list.

One pass solution (slow & fast pointers)

The slow & fast pointers pattern could be applied to solve this problem, the main process is described as below:

  1. Given slow and fast pointers pointing to the sentinel.
  2. Move the fast pointer n step forward for finding out the length of the list.
  3. Move both slow and fast pointers forward until the fast pointer travel to the end of the list.
  4. Delete the next element of the slow pointer to get the result.

Implementation

Two pass solution

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
  public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode cur = head;
    int len = 0;
    while (cur != null) {
      cur = cur.next;
      len++;
    }
    ListNode dummy = new ListNode(0), d = dummy;
    dummy.next = head;
    for (int i = 0; i < len - n; i++) {
      d = d.next;
    }
    d.next = d.next.next;
    return dummy.next;
  }
}

One pass solution (slow & fast pointers)

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
  public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0);
    dummy.next = head;
    ListNode slow = dummy, fast = dummy;
    for (int i = 0; i <= n; i++) {
      fast = fast.next;
    }
    while (fast != null) {
      slow = slow.next;
      fast = fast.next;
    }
    slow.next = slow.next.next;
    return dummy.next;
  }
}

Complexity

Two pass solution

Time: $O(n)$

Space: $O(1)$

One pass solution (slow & fast pointers)

Time: $O(n)$
Space: $O(1)$
