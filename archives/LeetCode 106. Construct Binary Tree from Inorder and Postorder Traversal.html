LeetCode 106. Construct Binary Tree from Inorder and Postorder Traversal

Problem

Source link: Construct Binary Tree from Inorder and Postorder Traversal

Given inorder and postorder traversal of a tree, construct the binary tree.

Note:
You may assume that duplicates do not exist in the tree.

For example, given

inorder = [9,3,15,20,7]
postorder = [9,15,7,20,3]

Return the following binary tree:

    3
   / \
  9  20
    /  \
   15   7

Design

The solving process of this problem is similar to Construct Binary Tree from Preorder and Inorder Traversal. We should first recall how inorder and postorder traversal look like: 

void inorderTraverse(TreeNode root) &#123;
  inorderTraverse(root.left);
  inorder.add(root.val);
  inorderTraverse(root.right);
&#125;
void postorderTravoerse(TreeNode root) &#123;
  postorderTraverse(root.left);
  postorderTraverse(root.right);
  postorder.add(root.val);
&#125;

Next, identify where the root node, left and right subtrees are.

By observing the places of those elements, we could come up with a pattern as below:

Implementation

/**
 * Definition for a binary tree node.
 * public class TreeNode &#123;
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() &#123;&#125;
 *     TreeNode(int val) &#123; this.val = val; &#125;
 *     TreeNode(int val, TreeNode left, TreeNode right) &#123;
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     &#125;
 * &#125;
 */
class Solution &#123;
  public TreeNode buildTree(int[] inorder, int[] postorder) &#123;
    return buildTree(inorder, postorder, 0, inorder.length - 1, 0, postorder.length - 1);
  &#125;

  private TreeNode buildTree(int[] inorder, int[] postorder,
                             int inStart, int inEnd, int postStart, int postEnd) &#123;
    if (inStart > inEnd || postStart > postEnd) &#123;
      return null;
    &#125;

    TreeNode root = new TreeNode(postorder[postEnd]);
    int idx = 0;
    for (int i = inStart; i <= inEnd; i++) &#123;
      if (inorder[i] == root.val) &#123;
        idx = i;
      &#125;
    &#125;

    int leftSize = idx - inStart;
    root.left = buildTree(inorder, postorder, inStart, idx - 1, postStart, postStart + leftSize - 1);
    root.right = buildTree(inorder, postorder, idx + 1, inEnd, postStart + leftSize, postEnd - 1);

    return root;
  &#125;
&#125;

Complexity

Time: $O(n)$

Space: $O(n)$
