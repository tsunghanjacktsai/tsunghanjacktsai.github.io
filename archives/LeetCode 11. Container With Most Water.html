LeetCode 11. Container With Most Water

Problem Statement

Source link: Container With Most Water

Given n non-negative integers a1, a2, …, an , where each represents a point at coordinate (i, ai). n vertical lines are drawn such that the two endpoints of line i is at (i, ai) and (i, 0). Find two lines, which together with x-axis forms a container, such that the container contains the most water.

Note: You may not slant the container and n is at least 2.

Example:

Input: [1,8,6,2,5,4,8,3,7]
Output: 49

Design

The problem can be simply solved by the two pointers pattern. The container size can be calculated as: (right_pos - left_pos) * min(left, right).

  1. Initialize left, right pointers for the array height and the variable tmp to store the result of container size calculated each time.
  2. If the height of left position is higher than the right, the container height would be the one where right pointer at then move the right pointer one step left and vice versa.
  3. The variable res would be assigned with the tmp or maintain the same value. Repeat 2 and 3 steps until left_pos = right_post.

Implementation

class Solution {
  public int maxArea(int[] height) {
    if (height.length == 0) return 0;
    int left = 0, right = height.length - 1, tmp = 0, res = 0;
    while (left < right) {
      if (height[left] > height[right]) {
        tmp = (right - left) * (height[right]);
        right--;
      } else {
        tmp = (right - left) * (height[left]);
        left++;
      }
      res = Math.max(res, tmp);
    }
    return res;
  }
}

Complexity

Time: $O(n)$

Space: $O(1)$
