27. Remove Element

Source link: https://leetcode.com/problems/remove-element/

Design

As the problem stated that the input array should be modified in-place with O(1) memory, extra space cannot be initiated for storing the data. To solve the problem, the two-pointer pattern can be applied for recording positions.

  1. Initialize two pointers i, j at the start of the array.
  2. Travel through the array leading by the pointer j and record the number preparing to be modified by the pointer i.
  3. Start the traverse process, once if the value of element is not equal to val, modify it by nums[j] and move the pointer i forward.

Implementation

class Solution {
  public int removeElement(int[] nums, int val) {
    int i = 0;
    for (int j = 0; j < nums.length; j++) {
      if (nums[j] != val) {
        nums[i++] = nums[j];
      }
    }
    return i;
  }
}

Complexity Analysis

Time: $O(n)$

Space: $O(1)$
