LeetCode 74. Search a 2D Matrix

Problem

Source link: Search a 2D Matrix

Write an efficient algorithm that searches for a value in an m x n matrix. This matrix has the following properties:

  • Integers in each row are sorted from left to right.
  • The first integer of each row is greater than the last integer of the previous row.

Example 1:

Input: matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,50]], target = 3
Output: true

Example 2:

Input: matrix = [[1,3,5,7],[10,11,16,20],[23,30,34,50]], target = 13
Output: false

Constraints:

  • m == matrix.length
  • n == matrix[i].length
  • 0 <= m, n <= 100
  • -104 <= matrix[i][j], target <= 104

Design

The key to solve this problem is to treat the 2D array as a 1D array. As the matrix size is m * n, we can transform it with the following formula:

2D: matrix[x][y] = 1D: matrix[x * n + 1]

1D: matrix[x] = 2D: matrix[x / n][x % n]

Here comes the example:

Now we can done the binary search on 1D version of the array as usual. Return the result if we find the target element.

Implementation

class Solution {
  public boolean searchMatrix(int[][] matrix, int target) {
    if (matrix == null || matrix.length == 0 || matrix[0].length == 0) {
      return false;
    }
    int n = matrix.length, m = matrix[0].length;
    int start = 0, end = n * m - 1;
    while (start <= end) {
      int mid= start + (end - start) / 2;
      if (target == matrix[mid / m][mid % m]) {
        return true;
      } else if (target < matrix[mid / m][mid % m]) {
        end = mid - 1;
      } else {
        start = mid + 1;
      }
    }
    return false;
  }
}

Complexity

Time: $O(logm * logn)$

Space: $O(1)$
