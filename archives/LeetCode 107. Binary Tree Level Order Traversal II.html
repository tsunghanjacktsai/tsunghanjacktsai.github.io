LeetCode 107. Binary Tree Level Order Traversal II

Problem

Source link: Binary Tree Level Order Traversal II

Given a binary tree, return the bottom-up level order traversal of its nodes’ values. (ie, from left to right, level by level from leaf to root).

For example:
Given binary tree [3,9,20,null,null,15,7],

    3
   / \
  9  20
    /  \
   15   7

return its bottom-up level order traversal as:

[
  [15,7],
  [9,20],
  [3]
]

Design

To solve this problem, we just need to simply change a little of the way we solved Binary Tree Level Order Traversal. Here’s the solving process:

  1. Given a queue for storing elements for the later use.
  2. Start doing BFS by reading tree’s root node into the queue.
  3. Get nodes in the current level from the queue.
  4. Store nodes in the next level into the queue if available.
  5. Record all nodes in the same level to the result list from its head (reverse the print order).

Implementation

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
  public List<List<Integer>> levelOrderBottom(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) {
      return result;
    }
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    while (!queue.isEmpty()) {
      List<Integer> curLevel = new ArrayList<>();
      int levelSize = queue.size();
      for (int i = 0; i < levelSize; i++) {
        TreeNode node = queue.poll();
        curLevel.add(node.val);
        if (node.left != null) {
          queue.offer(node.left);
        }
        if (node.right != null) {
          queue.offer(node.right);
        }
      }
      result.add(0, curLevel);
    }
    return result;
  }
}

Complexity

Time: $O(n)$

Space: $O(n)$ (queue size)
