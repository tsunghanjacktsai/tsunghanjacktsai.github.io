LeetCode 113. Path Sum II

Problem

Source link: Path Sum II

Given a binary tree and a sum, find all root-to-leaf paths where each path’s sum equals the given sum.

Note: A leaf is a node with no children.

Example:

Given the below binary tree and sum = 22,

      5
     / \
    4   8
   /   / \
  11  13  4
 /  \    / \
7    2  5   1

Return:

[
   [5,4,11,2],
   [5,8,4,5]
]

Design

This is an advanced follow-up problem of Path Sum. which can be solved by Tree DFS pattern. The process is described below:

  1. Traverse left and right subtrees and record each element on the path.
  2. If the target sum has been met, add the path into the result list.
  3. Remember to remove the last element of the path each time to enable backtracking.

Implementation

/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
  public List<List<Integer>> pathSum(TreeNode root, int sum) {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> curPath = new ArrayList<>();
    pathSum(root, sum, curPath, res);
    return res;
  }

  private void pathSum(TreeNode root, int sum, List<Integer> curPath, List<List<Integer>> res) {
    if (root == null) {
      return;
    }

    curPath.add(root.val);

    if (root.val == sum && root.left == null && root.right == null) {
      res.add(new ArrayList<>(curPath));
    } else {
      pathSum(root.left, sum - root.val, curPath, res);
      pathSum(root.right, sum - root.val, curPath, res);
    }

    curPath.remove(curPath.size() - 1);
  }
}

Complexity

Time: $O(n^2)$ (traverse each node once and store its path)

Space: $O(nlogn)$ (result list)
