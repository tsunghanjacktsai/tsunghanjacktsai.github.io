LeetCode 57. Insert Interval

Problem

Souce link: Insert Interval

Given a set of non-overlapping intervals, insert a new interval into the intervals (merge if necessary).

You may assume that the intervals were initially sorted according to their start times.

Example 1:

Input: intervals = [[1,3],[6,9]], newInterval = [2,5]
Output: [[1,5],[6,9]]

Example 2:

Input: intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]], newInterval = [4,8]
Output: [[1,2],[3,10],[12,16]]
Explanation: Because the new interval [4,8] overlaps with [3,5],[6,7],[8,10].

Example 3:

Input: intervals = [], newInterval = [5,7]
Output: [[5,7]]

Example 4:

Input: intervals = [[1,5]], newInterval = [2,3]
Output: [[1,5]]

Example 5:

Input: intervals = [[1,5]], newInterval = [2,7]
Output: [[1,7]]

Constraints:

  • 0 <= intervals.length <= 104
  • intervals[i].length == 2
  • 0 <= intervals[i][0] <= intervals[i][1] <= 105
  • intervals is sorted by intervals[i][0] in ascending order.
  • newInterval.length == 2
  • 0 <= newInterval[0] <= newInterval[1] <= 105

Design

  1. Given a list for storing result intervals, and an index pointing to each interval within the original intervals array.
  2. Traverse through the original intervals array, insert those intervals, which their upper bounds are smaller than the lower bound of new interval, into the result intervals list.
  3. Get the smaller one between lower bounds and the larger one between upper bounds of old and new intervals and add them into the result list.
  4. Insert remaining intervals into the result list.

Implementation

class Solution {
  public int[][] insert(int[][] intervals, int[] newInterval) {
    List<int[]> mergedIntervals = new ArrayList<>();
    int i = 0;

    while (i < intervals.length && newInterval[0] > intervals[i][1]) {
      mergedIntervals.add(intervals[i++]);
    }

    while (i < intervals.length && newInterval[1] >= intervals[i][0]) {
      newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
      newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
      i++;
    }
    mergedIntervals.add(newInterval);

    while (i < intervals.length) {
      mergedIntervals.add(intervals[i++]);
    }

    return mergedIntervals.toArray(new int[0][]);
  }
}

Complexity

Time: $O(n)$

Space: $O(n)$
